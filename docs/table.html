SubterfugeFrame

struct SubterfugeFrame Represents complete state for a Subterfuge App. Strictly speaking, none of the member values are required, however it is expected, that at least `list` or `form` is provided.

menu Array [ MenuItem ] If not provided, menu from previous frame might be reused.
list ItemList
form Form
title String "" optional. User-friendly title of this frame.

MenuItem

struct MenuItem

href String "posts" Unique identifier.
label String "Posts" User-friendly title.
icon String //. optional. Full URL to an svg/png/gif file. "" optional. Full URL to an svg/png/gif file.

ItemList

struct ItemList

filters Array [ ListFilter ]
headers Array [ ListHeader ]
items Array [ ListItem ]
pages Array [ ListPage ]
create_href String "posts/0" apu
filter_href String "posts" morbid

ListFilter

struct ListFilter

name String "title"

value String ""

ListFilter

struct ListFilter

ListHeader

ListItem

ListPage

struct ListPage

Form

struct Form

href String "posts/1"
title String "Post #1"
fields Array [ FormField ]
groups Array [ FormGroup ] optional. If specified, splits form fields into several groups.

FormField

struct FormField

name String "title" name of the field
value String "Post #1" current value
type String "text" enum FormWidget
options Array [ Option ] optional. Specifies options for multiple-choice widgets
group String "" optional. If set, must match FormGroup `name`.

Option

struct Option

value String "1"
label String "One"

value String "2"
label String "Two"

FormGroup

struct FormGroup Represents a fieldset. NOTE: FormField objects MUST have `group` value set.

label String "Main"
name String "main"
